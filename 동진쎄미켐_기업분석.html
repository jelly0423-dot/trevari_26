삼성전자의 두 가지 PR 공정 변화

① 즉각적 위협 (2024~)
NAND KrF PR 도포량 절반 감소
삼성 9세대 3D NAND부터 PR 도포량 7~8cc → 4~4.5cc로 50% 절감. 동진쎄미켐 PR 매출 2,500억 중 삼성 NAND향 60% = 약 1,500억이 영향권. 이론상 최대 750억 매출 감소 가능성. 2025년 매출은 오히려 증가했으나 점진적으로 반영 전망
② 중장기 구조 위협 (2026~)
삼성 1c D램 건식 PR 도입
삼성전자 6세대(1c) D램에 램리서치 건식 PR 업계 최초 적용(2025.06 확인). 기체 증착 방식으로 습식 PR 대비 소재 낭비 없음. 소재 공급자가 램리서치(장비사)로 동진쎄미켐 원천적 진입 불가. 양산은 2026년 본격화 예정
